Job Costing Application

This job costing tool is intended to be used by small contracting companies to allow them to provide live updates to their project stakeholders as well as closely track and monitor all expenses associated with a project.

This app will allow for the leads to see progress in the financial and day to day workload by tracking and visualizing budgeted labor expenses, material expenses, miscellaneous expenses and subcontractor expenses, and comparing them to the actual values. This limits the back and forth of chasing employees around for time sheets and daily progress and trying to remember what expenses were actually associated with the project. Knowing when a project is exceeding budgeted expenses is often the first step to preventing a project from becoming a major loss.

Horry County Flood Map

For this project, I was tasked with using data collected by canvassing, web scraping and collected from Horry County, South Carolina. This data looked out reported flooding events and compared that to flood zone data reported by FEMA. The intent of this study was to determine if current flood zones accurately informed consumers of their risk of flooding when purchasing houses in Horry County.

We used ArcGIS mapping software to show where reported flooding events occurred in the previous 10+ years. We plotted those converted addresses to latitude and longitude coordinates and plotted those to a map that showed current FEMA flood zones. This showed that there were many flood events in areas that according to FEMA were not in flood zones. This is a strong indicator that the FEMA flood zone maps need to be updated for this area.

We also prepared many data visualizations that showed everything from the amount of precipitation by weather station for a given number of years to showing how land usage had changed between 1996 and 2016.

eBirds_API

For this exercise, we made several api requests from Cornell University's Ebird API. In order to run these request you will need to obtain an eBird API Key.

Below is a list of API Request in order to get the data we needed for the visualizations shown below.

  • Migratory Pattern (Ruby-throated Hummingbird): https://api.ebird.org/v2/data/obs/US-TX/historic/2019/{month}/{day}
  • Notable Observations: https://api.ebird.org/v2/data/obs/
  • Endangered Birds: https://api.ebird.org/v2/data/obs/{{regionCode}}/recent, https://api.ebird.org/v2/product/stats/{{regionCode}}/{{y}}/{{m}}/{{d}}
  • ParaKeet Story: https://api.ebird.org/v2/data/obs/{{regionCode}}/historic/{{y}}/{{m}}/{{d}}
